# Privacy Policy

**MarkUp Markdown Editor**
*Last updated: June 2025*

## Overview

MarkUp Markdown Editor is a local, offline desktop application for Windows. Your privacy is important, and this application is designed with privacy in mind.

## Data Collection

MarkUp Markdown Editor does **not** collect, transmit, store, or share any personal data, usage data, analytics, or telemetry of any kind.

## Local Storage

All files you create, open, or edit remain on your local device. The application does not upload or sync files to any remote server or cloud service.

## Network Access

MarkUp Markdown Editor does not make network requests, except when rendering Markdown content that references external resources (e.g., images linked via URL). These requests are initiated solely by the content you provide and are not tracked or logged by the application.

## Third-Party Services

This application does not integrate with any third-party analytics, advertising, or tracking services.

## Changes to This Policy

If this privacy policy changes, the updated version will be published in this repository.

## Contact

If you have questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/John-Donnelly/MarkUp-Markdown-Editor).
